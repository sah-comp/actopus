Cinnebar

English

There might be something wrong with the Cinnebar installation unless you have entered the URL manually. If you did not expect to see this page, check the following stuff:

  • PHP may not be running
  • Apache mod-rewrite might be missing
  • The .htaccess file may be missing
  • File Access may be messed up

Deutsch

Prüfen Sie folgende Umstände falls Sie diese Seite nicht manuell aufgerufen haben:

  • PHP ist vielleicht nicht aktiviert
  • Der Webserver erlaubt keine URL Umschreibung
  • Die Datei .htaccess existiert nicht oder wird nicht ausgeführt
  • Lese- und Schreibrechte sind vielleicht nicht korrekt
